Django Cat Forum

Description

This is an interative multi-users forum for cat topics built by a cat-lover. It utilizes Django ModelForm with customized user model for authentication, and protects the main views. User database layer built with MySQL and Django ORM. What distinguishes this forum from catsite is it implement a "lost cat" dataset with Google Map APIfor cat adoption. It also looks fancy, unlike normal forum.

Technologies

Python, Django, MySQL, HTML5, Bootstrap, CSS3, JQuery, Github, Google Map API, JavaScript.

RaspberryPi Gameboy with Spaceship Game

Description

This is a group project we built during Coding Dojo. We installed a RaspberryPi B with customized screen in an old Gameboy. I wrote the spaceship game with JavaScrit using animation and OOP techniques, and HTML5, CSS3. User can select different spaceship toplay with. Enemies appears at random place, explodes when hit by the fireball. The most challenging is detecting object collision in 2D without lagging browser performance.

Technologies

JavaScript,HTML5,CSS3,Github,GIMP,Linux Raspian.
